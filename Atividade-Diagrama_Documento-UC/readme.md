# Demon Compendium

Pensando no desenvolvimento de seu próximo jogo, a equipe de desenvolvimento de jogos da Atlus decidiu criar um documento do que seria necessário na mecânica principal de seu jogo: o compêndio demoníaco.

Durante seu desenvolvimento, foi constado que seria necessário conter o jogador, que escolherá qual ação executar do compêndio, tais como fundir demônios criando novos demônios a partir de tipos de fusão diferentes (dupla [normal], tripla [triângulo], quadrupla [cruz], quintupla [pentágono] e sextupla [hexágono]), catalogar demônios não registrados no compêndio, invocar (comprar) demônios já catalogados, atualizar os status de um demônio já catalogado.
Para que possa ser possível executar as ações do compêndio, o jogador primeiro precisará entrar em um local específico, aqui definido como Velvet Room. As ações de fundir um demônio só poderão ser feitas caso o jogador escolha a função de criar novos demônios, assim falando com o líder da Velvet Room, que é o responsável por fundir os demônios. Em caso do jogador querer executar as outras funções, será necessário escolher a opção "demon compendium", assim falando com a assistente da Velvet Room.
